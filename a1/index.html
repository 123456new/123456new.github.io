手语识别项目 (a1)

基于 MediaPipe + BLSTM 的手语识别

文件结构

  • realtime_demo.py — 实时手语识别演示（MediaPipe Pose）
  • train.py — 训练脚本
  • SLR_flask_server.py — Flask 服务器
  • nnet/ — 神经网络模型（BLSTM/LSTM）
  • config/ — 配置文件
  • data/ — 数据集
  • model/ — 训练好的模型

运行方式

pip install -r requirements.txt.old
python realtime_demo.py
